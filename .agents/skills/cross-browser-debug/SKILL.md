# Cross Browser Debug

用于处理同一页面在不同浏览器中表现不一致的问题。

## 适用问题

- 某浏览器样式错位
- 某浏览器事件失效
- 滚动、选择、复制、下载行为不一致
- 新 API 在部分浏览器不可用
- 表单控件样式或行为差异

## 排查顺序

1. 先确认问题仅在某浏览器出现还是全部出现。
2. 对比 HTML 结构、样式类名、事件绑定与运行时分支。
3. 检查是否使用兼容性不足的 API。
4. 检查滚动容器、尺寸计算、选择范围、下载逻辑。
5. 给出兼容写法或降级方案。

## 常见检查点

- `scrollIntoView`、`Selection`、`Clipboard`、`FileReader`、`Blob`、下载链接行为
- `position: sticky`、`overflow`、flex / grid 尺寸计算
- 深色模式、原生表单控件、滚动条表现差异
- 键盘事件、输入法、快捷键差异

## 输出要求

- 哪个浏览器异常
- 兼容性根因
- 建议修复方式
- 验证浏览器列表

## 边界

- 优先采用简单稳定写法。
- 不为了少数边缘效果引入重型 polyfill。
