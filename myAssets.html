表格/JSON 格式说明（导入）

Excel：第一行表头支持下列任一列名（中英皆可，大小写不敏感）：

  • name / 名称（必填）
  • type / 类型（选填，默认“其他”）示例：股票、基金、现金
  • value / 金额（必填，数字）
  • currency / 币种（选填，默认 CNY）
  • remark / 备注（可选）

Excel 示例表头：

name,type,value,currency,remark
招商中证白酒ETF,基金,20000,CNY,长期持有
贵州茅台,股票,35000,CNY,核心持仓

JSON：支持两种结构：

[
  { "name": "招商中证白酒ETF", "type": "基金", "value": 20000, "currency": "CNY", "remark": "长期持有" },
  { "name": "贵州茅台", "type": "股票", "value": 35000, "currency": "CNY", "remark": "核心持仓" }
]

{
  "assets": [
    { "name": "招商中证白酒ETF", "type": "基金", "value": 20000, "currency": "CNY", "remark": "长期持有" }
  ]
}

校验规则：name 必填，value 必须为数字；其余字段可选（type 默认“其他”、currency 默认 CNY）。导入时会自动忽略缺少名称或金额非数字的行，并提示统计结果。
