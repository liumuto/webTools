# 🚀 量化选股系统启动指南

## 📋 系统概述

本系统是一个基于技术指标和图形识别的智能选股平台，包含：
- **后端API服务**：提供股票数据获取、技术分析、选股策略等接口
- **前端Web界面**：用户友好的可视化操作界面
- **量化选股引擎**：8种基础策略 + 6种图形识别算法

## 🔧 启动方法

### 方法1：一键启动（推荐）

```bash
# 1. 进入项目目录
cd src/stocks

# 2. 运行启动脚本
python start_quantitative.py
```

启动脚本会自动：
- ✅ 检查依赖包
- ✅ 验证必要文件
- ✅ 启动后端API服务
- ✅ 显示访问地址

### 方法2：手动启动

#### 启动后端服务
```bash
cd src/stocks
python app.py
```

#### 访问前端界面
打开浏览器访问：`src/stocks/ui/myStock.html`

### 方法3：分步启动

#### 1. 检查环境
```bash
cd src/stocks
python -c "import flask, pandas, numpy, akshare; print('环境检查通过')"
```

#### 2. 启动后端
```bash
python app.py
```

#### 3. 打开前端
在浏览器中打开：`ui/myStock.html`

## 🌐 访问地址

- **API文档**: http://localhost:5000/api/stocks/strategies
- **健康检查**: http://localhost:5000/api/stocks/health

### 前端界面
- **文件路径**: `src/stocks/ui/myStock.html`

## 📊 API接口说明

| 接口 | 方法 | 描述 | 示例 |
|------|------|------|------|
| `/api/stocks/list` | GET | 获取股票列表 | `GET /api/stocks/list` |
| `/api/stocks/analyze/<code>` | GET | 分析单只股票 | `GET /api/stocks/analyze/000001` |
| `/api/stocks/select` | POST | 批量选股 | `POST /api/stocks/select` |
| `/api/stocks/strategies` | GET | 获取策略说明 | `GET /api/stocks/strategies` |
| `/api/stocks/health` | GET | 健康检查 | `GET /api/stocks/health` |

## ⚠️ 常见问题

### 1. 端口被占用
```bash
# 查看端口占用
netstat -an | findstr :5000

# 杀死占用进程
taskkill /f /pid <进程ID>
```

### 2. 依赖包缺失
```bash
# 安装核心依赖
pip install flask flask-cors pandas numpy akshare

# 安装可选依赖（绘图功能）
pip install matplotlib seaborn
```

### 3. 数据获取失败
- 检查网络连接
- 确认AKShare服务正常
- 查看控制台错误信息

### 4. 前端无法访问后端
- 确认后端服务已启动
- 检查CORS设置
- 确认API地址正确

## 📈 使用流程

1. **启动系统**：运行 `python start_quantitative.py`
2. **打开前端**：在浏览器中打开 `ui/myStock.html`
3. **配置策略**：选择股票池、时间范围、选股策略
4. **执行选股**：点击"开始选股"按钮
5. **查看结果**：分析选股结果和评分
6. **导出数据**：保存选股结果到文件
