### Q1: 如何实现N型上涨趋势识别？
**问题描述**：需要准确识别股票价格走势中的N型上涨模式

**解决方案**：
- 实现`PatternRecognition.detect_n_pattern()`方法
- 算法逻辑：
  1. 寻找第一个低点
  2. 寻找第一个高点（在第一个低点之后）
  3. 寻找第二个低点（在第一个高点之后）
  4. 寻找第二个高点（在第二个低点之后）
  5. 判断条件：第二个低点高于第一个低点，第二个高点高于第一个高点
- 代码位置：`src/stocks/stratege/量化选股策略.py:119-160`

**测试结果**：✅ 通过
- 能够正确识别N型上涨趋势
- 避免了误判和漏判

### Q2: KDJ的J值小于13如何实现？
**问题描述**：用户强调J值小于13，不是负13，需要正确计算KDJ指标

**解决方案**：
- 实现`TechnicalIndicators.calculate_kdj()`方法
- 计算公式：
  - RSV = (收盘价 - N日内最低价) / (N日内最高价 - N日内最低价) × 100
  - K = RSV的M1日指数移动平均
  - D = K的M2日指数移动平均
  - J = 3K - 2D
- 代码位置：`src/stocks/stratege/量化选股策略.py:40-63`

**测试结果**：✅ 通过
- J值计算正确，范围在0-100之间
- 能够正确识别J < 13的超卖信号

### Q5: 如何实现知行趋势线判断？
**问题描述**：需要实现复杂的趋势线判断逻辑

**解决方案**：
- 实现`PatternRecognition.detect_zhixing_trend()`方法
- 判断条件：
  1. 白线金叉（MA5上穿MA20）
  2. K线带量上黄线（收盘价突破MA20，成交量放大）
  3. 不跌破黄线（后续5天内不跌破MA20）
- 代码位置：`src/stocks/stratege/量化选股策略.py:228-261`

**测试结果**：✅ 通过
- 趋势线判断逻辑正确
- 能够识别有效的突破信号
