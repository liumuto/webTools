# 产品需求文档：个人资产分布管理与可视化系统

## 一、产品概述

系统用于统计和可视化个人资产，包括 **股票、基金、现金** 等类别。用户可以：

* 从 Excel 表格导入数据
* 通过 JSON 粘贴/文件 导入数据
* 对资产进行增删查改
* 使用矩形网格分块、不同颜色表示资产分布
* 支持全局展示和分类筛选

技术栈：**HTML + CSS + 原生 JavaScript**

---

## 二、核心功能

### 1. 数据管理

* **数据来源**

  * Excel 导入（使用 [SheetJS](https://github.com/SheetJS/sheetjs) 解析 `.xlsx` 文件）
  * JSON 文件导入（解析 `.json` 文件）
  * JSON 粘贴导入（在页面文本框粘贴 JSON 文本）
  * 手动录入（表单输入）
* **数据结构（内存中存储）**

  ```js
  {
    id: "uuid-001",
    name: "招商中证白酒ETF",
    type: "基金",
    value: 20000,
    currency: "CNY",
    remark: "长期持有"
  }
  ```

* **操作**

  * 新增：通过表单提交数据
  * 修改：在表格中编辑数据
  * 删除：点击删除按钮
  * 查询：按类型、金额区间、关键字搜索

### 2. 数据展示（可视化）

* **全局视图**

  * 使用 Treemap（有序条带 Strip Treemap），二级布局：先按类别（type）分配顶层区域，再在每个类别区域内按标的 `value` 铺排
  * 各资产块面积严格按金额（value）占比计算；位置规则：大的在左上，小的在右下
  * 不同资产类型用不同颜色
  * 小块纵向列展示：当剩余资产块的预估尺寸小于最小像素阈值（如 24×16 px）时，启用“纵向列展示”（slice-and-dice 垂直切分），将小额资产按金额从大到小以竖列排列，保证完整显示
  * 列布局自适应：竖列可根据可视区域宽度换列（多列），不足一列的部分自动换行；超出可视高度时列内可滚动
  * 小额聚合：支持将极小金额合并为“其他”分组，点击可展开/收起
* **筛选视图**

  * 按类型（股票/基金/现金…）
  * 按金额区间
  * 按关键字

### 3. 交互功能

* **悬停** → 显示 tooltip（资产详情）
* **点击块** → 弹出编辑/删除操作窗口
* **列视图交互**：

  * 列内滚动：当列内容超出高度时，独立滚动不影响主视图
  * 展开/收起“其他”分组：点击展开为列视图
  * 小块 hover 高亮并在列表中联动高亮
* **全局操作栏**

  * 导入 Excel（文件选择框 + JS解析）
  * 导入 JSON（文件选择框）
  * 粘贴 JSON 导入（文本输入区 + 导入按钮）
  * 导出数据（JSON/CSV/Excel）
  * 筛选器（下拉框 + 输入框）

### 页面结构

1. **顶部工具栏**

   * \[导入Excel按钮]

## 四、技术方案

### 前端实现

* **HTML**：结构布局（工具栏、表格、画布区域）
* **CSS**：

  * Flex/Grid 布局
  * Treemap 方块颜色区分
  * Tooltip 悬浮样式
* **JavaScript**：

  * 数据存储：数组（全局变量）
  * 数据增删查改（原生 DOM 操作）
  * 数据导入导出：

    * Excel 导入导出：SheetJS（仅引入单文件 js）
    * JSON 解析：支持文件读取与文本粘贴两种方式
  * Treemap 计算与渲染：

    * 按资产金额比例计算方块大小
    * 使用有序条带（Strip Treemap）算法：
      * 顶层：按类别总金额划分区域，自上而下、从左到右铺排
      * 类别内：children 按 value 降序，横向条带逐行铺排；较大在左上，较小在右下
      * 不强求方正，允许长条矩形；面积严格按 value 比例
    * 小额聚合与“其他”组逻辑：小于阈值的资产先聚合为“其他”，支持点击展开进入列视图
    * 使用 `<canvas>` 或 `<div>` 容器渲染（列区支持独立滚动容器）

## 五、用户场景示例

1. 用户导入 Excel → 表格展示 → Treemap 自动更新
2. 用户粘贴/导入 JSON → 表格展示 → Treemap 自动更新
3. 数据量较多 → Treemap 自动在底部进入“纵向列展示/其他分组” → 列内滚动查看完整小额项
4. 用户筛选 “基金” → 表格和图表仅显示基金数据
5. 用户点击某个方块 → 弹出编辑表单 → 修改金额 → 页面自动更新
6. 用户点击导出 → 下载 JSON/CSV/Excel 文件
